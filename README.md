# Moniker cookbook

## Cookbook Development

Requirements:

1. Vagrant - http://www.vagrantup.com/
2. Virtualbox - https://www.virtualbox.org/
3. Berkshelf - http://berkshelf.com/
4. Vagrant Berkshelf plugin - https://github.com/riotgames/berkshelf-vagrant

Process:

1. `vagrant up`
2. Code..
3. `vagrant provision`
4. Goto 2
